2 Summary

e.Simple Notes is a Chrome side panel extension that provides notes, calculator, timer, AI chat, theme, language, and notification preferences.

No personal data sold No advertising SDKs No third-party analytics No developer-side data collection AI chat: BYOK, no developer access

3 Information the Extension Stores

The Extension may store the following in Chrome extension storage on the user's browser profile:

  • Notes created by the user
  • Note history and editor content
  • Timer session state (remaining time, running / paused / finished status)
  • App preferences such as language and notification settings
  • Theme preference stored in browser local storage
  • Note labels created by the user
  • Selected AI provider preference (stored locally only)
  • User-provided AI API key (stored in Chrome local extension storage on the user's own device only; never synced to Chrome sync storage or transmitted to the developer)

The Extension includes an optional user-controlled sync setting. When the user enables this setting, preferences (excluding the AI API key) may also be mirrored to Chrome sync storage so that the extension can restore preferences more reliably across devices. The AI API key is never written to Chrome sync storage regardless of this setting.

4 How the Information Is Used

The stored information is used only to operate the Extension's features, including:

  • saving and restoring notes and note labels
  • restoring timer state
  • showing timer completion notifications
  • remembering language, theme, and notification preferences

5 No Developer-Side Data Collection

The Extension does not transmit notes, timer data, settings, or any other stored content to the developer's own servers.

The Extension does not collect account information, passwords, payment information, precise location, health data, browsing history, or web page content on behalf of the developer.

When the optional AI chat feature is used, the user's messages are sent directly from the browser to the selected AI provider's API endpoint. This communication occurs between the user's browser and the AI provider. The developer does not receive, log, or process any of this data. See Section 6 for details.

6 AI Chat: Bring Your Own Key (BYOK)

The Extension includes an optional AI chat panel. To activate it, the user must provide their own API key from a supported provider. The following providers are currently supported:

  • OpenAI (api.openai.com)
  • Anthropic (api.anthropic.com)
  • Google Gemini (generativelanguage.googleapis.com)
  • DeepSeek (api.deepseek.com)

The following applies to this feature:

  • The user's API key is stored only in Chrome local extension storage on the user's own device. It is never sent to the developer's servers.
  • When the user sends a message, that message (and conversation history for the current session) is transmitted directly from the user's browser to the selected AI provider's API endpoint using the user's own API key.
  • If the user explicitly enables the "Include page context" toggle, the visible text of the currently active browser tab is extracted and appended to the outgoing request. This page text is sent to the AI provider, not to the developer.
  • Conversation history is kept in memory for the current session only and is not persisted to any storage after the session ends.
  • The developer does not receive, intercept, log, or store any chat messages, API keys, or page content.

Each AI provider's own privacy policy governs how that provider handles the data it receives:

  • OpenAI: openai.com/policies/privacy-policy
  • Anthropic: anthropic.com/privacy
  • Google: policies.google.com/privacy
  • DeepSeek: cdn.deepseek.com/policies/en-US/deepseek-privacy-policy.html

7 Permissions

The Extension uses the following Chrome permissions only for product functionality:

  • storage: to save notes, timer state, and settings
  • unlimitedStorage: to allow local note storage without tight default quota limits
  • alarms: to run the timer reliably
  • notifications: to show timer completion alerts
  • offscreen: to play timer alarm audio when needed
  • sidePanel: to run the extension inside the Chrome side panel
  • activeTab: to identify the currently active browser tab when the user explicitly enables the "Include page context" feature in AI chat
  • scripting: to extract the visible text of the current page when the user explicitly enables the "Include page context" feature in AI chat

The Extension also declares the following host permissions, which are required solely to allow the browser to communicate with user-selected AI provider APIs and to support the page context feature:

  • <all_urls>: required to allow the scripting API to read the visible text of the currently active browser tab when the user enables the "Include page context" feature. Because the Extension runs inside a Chrome side panel rather than a standard browser action popup, the activeTab permission alone does not grant on-demand scripting access to the active tab; a broad host permission is therefore necessary. No background browsing, passive tab monitoring, or data collection occurs.
  • https://api.openai.com/*
  • https://api.anthropic.com/*
  • https://generativelanguage.googleapis.com/*
  • https://api.deepseek.com/*

8 Data Sharing

The Extension does not sell, rent, or share personal information with data brokers or advertisers. The Extension does not share stored note content with third parties, except where Chrome itself processes extension storage as part of the browser platform.

When the user chooses to use the AI chat feature and sends a message, that message is shared with the AI provider selected by the user (OpenAI, Anthropic, Google Gemini, or DeepSeek). This sharing is initiated solely by the user's own action and is governed by the respective provider's privacy policy.

9 Data Retention and User Control

Data remains stored until the user edits, deletes, resets, or uninstalls the Extension. Users can remove stored notes and preferences from within the Extension or by removing the Extension from Chrome.

10 Security

The Extension is designed to keep user content inside the browser storage environment. However, no method of storage or transmission can be guaranteed to be 100% secure.

11 Children's Privacy

The Extension is not directed to children under 13, and the developer does not knowingly collect personal information from children.
